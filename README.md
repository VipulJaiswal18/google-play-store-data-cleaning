📊 Google Play Store Data Cleaning Project

📌 Project Overview

This project focuses on cleaning and preprocessing the Google Play Store dataset. The dataset contains missing values, duplicate entries, and inconsistent data. The main objective is to transform raw data into a clean and structured format suitable for analysis.

---

📁 Dataset Information

- Raw Dataset: Original dataset containing errors and inconsistencies
- Cleaned Dataset: Processed dataset after applying data cleaning techniques

---

🧹 Data Cleaning Process

The following steps were performed:

- Removed duplicate rows to ensure data uniqueness
- Handled missing values using appropriate techniques (mean, mode, or removal)
- Fixed corrupted and misaligned rows
- Converted data types (e.g., Reviews, Size, Installs) into proper formats
- Cleaned columns with inconsistent values (e.g., Size, Price, Installs)
- Standardized categorical data
